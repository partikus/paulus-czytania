== Description ==

Ta wtyczka pozwala na dodanie widgetu, który wyświetla cytat, czytania na dany dzień oraz sigla biblijne.
Wszystkie wyniki zwracane przez stronę http://www.edycja.pl/quotes są cachowane/przechowywane w bazie, tak aby import danych odbywał się tylko raz.


== Installation ==

Zainstaluj wtyczkę poprzez instalator na twojej stronie.
